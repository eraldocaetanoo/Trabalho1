# Trabalho1

Baixe os códigos no endereço https://github.com/eraldocaetanoo/Trabalho1.git;
Em seguida abra o NetBeans, clique na aba Arquivo e depois clique em Abrir projeto (ou use o atalho Ctrl+Shift+O);
Encontre o projeto baixado e clique em Abrir projeto;

Ao executar o programa, responda o que for pedido e a cada resposta tecle Enter;
Quando chegar na tela de operações escolha a operação desejada usando o mouse;
Siga os passos dados pelo programa.
